# 高品質程式碼生成（XML 結構）

```
<context>
語言：[替換：TypeScript]
框架：[替換：Next.js 14]
規範：[替換：ESLint + Prettier]
</context>

<task>
寫一個 [替換：功能描述]。
要求：
1. 包含完整的 TypeScript 型別
2. 包含錯誤處理
3. 包含 JSDoc 註解
4. 包含單元測試（使用 [替換：Jest]）
</task>

<output_format>
先列出檔案結構，然後逐一輸出每個檔案的完整程式碼。
</output_format>
```
